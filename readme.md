## Vistagram

A instagram alike project

### Run project

Frontend:

```
$ cd frontend
$ npm run start
```

Backend:

```
$ install docker
$ docker run --name some-postgres -e POSTGRES_PASSWORD=adminaccess4postg -d postgres

```

## Tech Stack

**Client:** React, Styled Component

**Backend:** Flask, Postgres
